  🚀 如何使用 Autoresearch 自动优化

  现在你可以让 Claude 自动优化模型了！运行：

  ===/autoresearch
  Goal: Maximize model efficiency - achieve >98% accuracy with minimal model size
  Scope: models/model.py
  Metric: Score (higher is better, balances accuracy and size)
  Verify: python evaluate.py | grep "Score"
  Guard: python train.py && python evaluate.py=== 

  🎯 Autoresearch 会做什么

  Claude 会自动：

  1. 分析当前模型 - 理解架构和改进空间
  2. 修改模型 - 尝试优化方向：
    - 减少卷积核数量（32→16, 64→32）
    - 减少全连接层维度（128→64）
    - 添加 Batch Normalization
    - 尝试深度可分离卷积
    - 调整 Dropout 比例
  3. 训练新模型 - 运行 python train.py
  4. 评估改进 - 运行 python evaluate.py
  5. 决策 - Score 提升 → 提交；Score 下降 → 回滚
  6. 持续迭代 - 不断优化直到达到目标或手动中断
